# 🎮 Tic-Tac-Toe

A modern and interactive **Tic-Tac-Toe** game built with **HTML, CSS, and JavaScript**. This project demonstrates fundamental web development concepts, including DOM manipulation, event handling, game logic, and responsive UI design.

## 🚀 Live Demo

🔗 https://abdullahalicoco123-debug.github.io/tic-tac-toe/

---

## ✨ Features

- 🎮 Two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset Game functionality
- 🆕 Start a New Game instantly
- 🎉 Winner announcement with modern UI
- 📱 Fully responsive design
- ⚡ Fast, lightweight, and easy to use

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

```bash

```

### Run the project

Open the `index.html` file in your preferred web browser.
